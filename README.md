# Genetic Dots

![Imgur](http://i.imgur.com/8XlNbmY.gif)

- **Language:** Python

A simple genetic algorithm where a bunch of dots try to find their way to the target.

## Getting Started


### Prerequisites

- Python +3.5
- tkinter (installed by default), if not :

```
apt-get install python-tk
```

### Launching


From root

```
python3 ./main.py
```
